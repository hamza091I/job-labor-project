Jo-Labor-AI Agent is an intelligent legal assistant tailored for the Jordanian job market. It utilizes Retrieval-Augmented Generation (RAG) to bridge
the gap between complex legal documents and everyday inquiries. Built with Python 3.13 and LangChain Expression Language (LCEL), it ensures that every
answer—whether regarding employment termination, leave entitlements,
or worker rights—is cited directly from official sources, eliminating AI hallucinations and providing reliable, document-backed insights.
